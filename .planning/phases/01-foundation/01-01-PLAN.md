Install Supabase packages:
`npm install @supabase/supabase-js @supabase/ssr`

Create Supabase client utilities following official patterns:

1. Create src/lib/supabase/client.ts:
   - Browser client using createBrowserClient from @supabase/ssr
   - Export function createClient() that returns browser Supabase client

2. Create src/lib/supabase/server.ts:
   - Server client using createServerClient from @supabase/ssr
   - Export function createClient() for use in Server Components and Route Handlers
   - Handle cookies properly for Next.js App Router

3. Create .env.local.example with:
   - NEXT_PUBLIC_SUPABASE_URL=your-project-url
   - NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key

Use @supabase/ssr (not @supabase/auth-helpers-nextjs which is deprecated).
Follow the official Supabase + Next.js App Router guide patterns.

Do NOT set up auth yet - that's the next plan. Just the base client configuration.
  </action>
